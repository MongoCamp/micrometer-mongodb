
# MongoCamp 

![](docs/public/logo_with_text_right.png)

## Introduction
Binder and Registry for Micrometer and MongoDB, to persist your [Micrometer](https://micrometer.io/) Stats to an MongoDB or monitor you MongoDB with your app.
